## Getting Started

clone the repo then `npm install` followed by `npm run dev`

### Prerequisites

- Node 8.11
- mongodb 3.6

### Installation

1. clone repo
2. copy dotenv sample file
2. install packages (client & server)
3. seed db
4. run dev
